Tamaño de caja

Es una propiedad de CSS que se utiliza para afectar como funciona el modelo de caja predeterminado

  • content-box: sólo se aplica al contenido.
  • border-box: el tamaño de la caja continene el contenido, relleno y borde. El contenido se adapta al tamaño de la caja.

Ejemplo

content-box: sólo se aplica al contenido. El contenido es largo en esta sección para tener ejemplo de como se ve.

border-box: El contenido se adapta al tamaño de la caja. El contenido es largo en esta sección para tener ejemplo de como se ve.
